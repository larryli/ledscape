# 硬件设计文档

## 1. 系统概述

LEDScape 风筝 LED 灯效系统，基于 CH32V003 驱动 30 颗 WS2812B 灯珠，3.7V 锂电池直供电。

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  LiPo 3.7V  │───→│  主控板      │───→│  灯带 1米    │
│  400mAh     │    │  CH32V003   │    │  30×WS2812B │
└─────────────┘    └─────────────┘    └─────────────┘
```

### 2.5 调试接口

3 个通孔，间距 2.54mm，用于 SWD 烧录调试：

| 通孔 | 功能 | 连接 |
|------|------|------|
| H1 | SWCLK | PD1 |
| H2 | SWDIO | PD3 |
| H3 | GND | VSS |

## 3. 灯带

### 3.1 规格

| 参数 | 值 |
|------|-----|
| 型号 | WS2812B |
| 数量 | 30 颗 |
| 长度 | 1 米 (每颗间距 33mm) |
| 供电电压 | 3.7V (直供) |
| 数据速率 | 800kHz |
| 最大电流 | 1800mA (全白) |
| 典型电流 | 400-600mA (彩虹效果) |

### 3.2 接线

灯带引出 3 根线，直接焊接到主控板焊盘：

| 灯带线 | 颜色 | 连接 |
|--------|------|------|
| VCC | 红 | 主控板 [+] |
| GND | 黑 | 主控板 [-] |
| DIN | 白/绿 | 主控板 [DIN] (PC6) |

### 3.3 安装方式

- 灯带沿风筝横梁骨架布置
- 用透明热缩管包裹防水
- 扎带固定在骨架上，间距 10cm

## 4. 电池

### 4.1 规格

| 参数 | 值 |
|------|-----|
| 类型 | LiPo 软包 |
| 型号 | 601010 |
| 容量 | 400mAh |
| 电压 | 3.7V (3.0V~4.2V) |
| 重量 | ~9g |
| 接口 | JST-PH 2.0mm 2pin |

### 4.2 续航估算

| 效果类型 | 典型电流 | 续航 |
|----------|----------|------|
| 彩虹流水 | ~600mA | 40 分钟 |
| 呼吸/烛光 | ~300mA | 80 分钟 |
| 混合播放 | ~450mA | 53 分钟 |

### 4.3 供电方式

```
电池 JST-PH 公头 ──→ 主控板 JST-PH 母座
                         │
                     拨动开关
                         │
                    CH32V003 + 灯带
```

3.7V 直供，无需稳压器。

## 5. 重量汇总

| 组件 | 重量 |
|------|------|
| 主控板 (含元器件) | ~1.2g |
| 灯带 (1米30颗) | ~18g |
| 电池 (400mAh) | ~9g |
| 线材 (30AWG 特氟龙) | ~4g |
| 固定材料 (热缩管/扎带) | ~3g |
| **总计** | **~35g** |

## 6. 防护设计

| 措施 | 方法 |
|------|------|
| 防水 | 灯带套热缩管，焊点涂硅胶 |
| 防风 | 所有连接用热熔胶加固 |
| 防震 | 电池用泡棉包裹 |
| 防脱落 | 灯带用扎带绑在骨架上 |

## 7. 风筝安装

### 7.1 布置方式

```
风筝骨架
├──────────────────────────────────┤
│ ◉◉◉◉◉◉◉◉◉◉◉◉◉◉◉◉◉◉◉◉◉◉◉◉◉◉◉◉◉◉ │  ← 灯带沿横梁布置
│          ↑                      │
│       主控板+电池               │
│       (中心位置)                │
└──────────────────────────────────┘
```

### 7.2 安装要点

- 灯带沿风筝横梁骨架布置
- 主控板和电池固定在风筝中心（保持平衡）
- 线材沿骨架走线，用扎带固定
- 所有接口涂硅胶防水

## 8. 飞行建议

- 风力：2-3 级微风
- 场地：开阔地，远离电线
- 首次试飞：地面测试平衡性
- 电池固定牢靠，防止脱落
