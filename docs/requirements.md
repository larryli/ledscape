# 需求文档

## 1. 项目概述

### 1.1 项目名称
LEDScape

### 1.2 项目目标
基于 CH32V003 微控制器，驱动 WS2812B RGB LED 灯带，内置 60 种远距离优化灯效，上电自动洗牌随机播放。

### 1.3 项目范围
- 硬件平台：CH32V003 + WS2812B LED 灯带 (30 颗，1 米)
- 软件架构：SPI + DMA 驱动 + 60 种灯效算法
- 支持封装：TSSOP-20 / QFN-20 / SOP-16

## 2. 功能需求

### 2.1 核心功能

#### F-01: WS2812B 驱动
- 使用 SPI1 + DMA1 Channel3 实现 WS2812B 数据输出
- 输出引脚：PC6 (SPI1 MOSI)
- SPI 时钟：3MHz (48MHz / 16)
- 双缓冲机制：DMA 半传输/传输完成中断交替填充缓冲区
- CPU 占用：~6%

#### F-02: 灯效算法
- 60 种灯效，分为 7 大类：
  - 彩虹类 (8 种)：流水、呼吸、脉冲、弹跳、剧场、擦除、波浪
  - 呼吸/脉冲类 (6 种)：暖白、冰蓝、红/绿/蓝脉冲、彩色闪烁
  - 流动/追逐类 (6 种)：彗星、流星雨、擦除、剧场追逐
  - 自然模拟类 (5 种)：烛光、极光、日落、海浪、火焰
  - 特殊效果类 (5 种)：闪烁、雪花、双色、三色、渐变
  - 远距离效果 (20 种)：螺旋、彗星彩虹、烟花、涟漪、脉冲环等
  - 补充效果 (10 种)：双彩虹、静态彩虹、暖光、冷光等

#### F-03: 自动切换
- 上电后自动洗牌随机播放 60 种灯效
- 每种灯效展示 3 秒
- 一轮播放完毕后重新洗牌
- 无需用户交互

### 2.2 辅助功能

#### F-04: LED 数量配置
- 编译时配置 LED 数量 (NR_LEDS)
- 默认值：30
- 范围：1-191

#### F-05: 颜色顺序配置
- 支持 RGB / GRB / RBG 颜色顺序
- 通过编译宏配置 (WSRAW / WSRBG / WSGRB)
- 默认值：WSGRB (适用于 SK6812)

#### F-06: 测试模式
- 支持编译时指定测试单个灯效 (TEST_EFFECT=N)
- 测试模式下禁用洗牌和切换逻辑

## 3. 硬件需求

### 3.1 主控芯片
- 型号：CH32V003
- 封装：TSSOP-20 (F4P6) / QFN-20 (F4U6) / SOP-16 (A4M6)
- 核心：QingKe V2A (RV32EC)
- 主频：48MHz (HSI 24MHz × PLL 2)
- Flash：16KB
- RAM：2KB

### 3.2 LED 灯带
- 型号：WS2812B / SK6812
- 数量：30 颗
- 长度：1 米
- 供电电压：5V (数据线兼容 3.3V)
- 数据传输速率：800kHz
- 每个 LED 功耗：最大 60mA (全白)

### 3.3 引脚分配

| 引脚 | 功能 | 方向 | 说明 |
|------|------|------|------|
| PC6 | SPI1 MOSI | 输出 | WS2812 数据线 |
| PC5 | SPI1 SCK | 输出 | SPI 时钟 (可选) |
| PD1 | SWCLK | - | 调试接口 |
| PD3 | SWDIO | - | 调试接口 |
| VDD | 电源 | 输入 | 3.3V |
| VSS | 地 | - | GND |

### 3.4 电源设计
- 芯片供电：3.3V (LDO)
- LED 灯带供电：5V (独立供电)
- 去耦电容：100nF (VDD-GND)

## 4. 性能需求

### 4.1 实时性
- 帧率：~60 FPS
- DMA 传输延迟：≤ 1ms
- 效果切换：每 3 秒自动切换

### 4.2 资源占用
- CPU 占用：~6% (输出期间)
- Flash 占用：7.3KB (44.58%)
- RAM 占用：232B (11.33%)

### 4.3 可靠性
- 连续运行时间：≥ 24 小时无异常
- LED 刷新：无闪烁、无颜色错误

## 5. 约束条件

### 5.1 硬件约束
- RAM 限制：2KB (当前使用 232B)
- 无硬件乘法器：使用移位优化乘法
- DMA 通道固定：SPI1 TX 只能使用 DMA1 Channel3
- GPIO 限制：单 IO 口方案只能使用 PC6

### 5.2 软件约束
- 驱动库：ch32fun SDK
- 开发语言：C
- 编译器：GCC RISC-V
